Overview:
¶

A supervised machine learning algorithm that uses a decision tree to generate classification and regression models. The attributes are used to classify data in the decision tree. The tree is made up of decision nodes and decision leaves, with each node having two or more branches that indicate the value of the attribute being examined. The leaf nodes generate a homogeneous result. The ID3 algorithm follows a greedy approach by selecting a best attribute that provides maximum Information Gain or minimum Entropy.


Approach:
¶

To find the entropy for the provided training dataset, we first constructed the entropy calculation function. We used the entropy calculation function to calculate the information gain once we found the entropy. The donut chart was then produced using the entropy and information gain values to illustrate the average number of positively and negatively responding patients for the new treatment. We also utilized a horizontal stacked bar chart to display the probability of positive and negative oracle for each attribute. By evaluating our program to the DecisionTreeClassifier() function in scikit.learn, we can ensure that it worked correctly on the Treatment dataset.

def entropyCalculation (dataset):
    val = []
    entropy = 0
    graph = []
    uniqueValues = dataset.iloc[:,-1].unique()

    for uniqueVal in uniqueValues:
        totVal = 0
        temp = 0
        for target in dataset.iloc[:,-1]:
            if (target == uniqueVal):
                temp += 1
            totVal += 1
        val.append(temp)

    for value in range(0,len(uniqueValues)):
        temp = ( val[value] / totVal )
        graph.append(temp * 100)
        entropy -= temp * math.log2(temp)

 
    my_circle = plt.Circle( (0,0), 0.7, color='white')
    plt.pie(graph, labels=uniqueValues, autopct='%1.0f%%', pctdistance=1.1, labeldistance=1.2)
    p = plt.gcf()
    p.gca().add_artist(my_circle)
    plt.show()
    
    return entropy

def attributeCalculation (dataset):
    attributeDict = {}
    truncData = fullObj
    truncData = truncData.iloc[:, :-1]
    for val in fullObj:
        lastColumn = val
    for column in truncData:
        uniqueValues = fullObj[column].unique()
        uniqueTarget = fullObj.iloc[:,-1].unique()
        attributeDict[column] = {}
        for uniqueVal in uniqueValues:
            attributeDict[column][uniqueVal] = {}
            for unique in uniqueTarget:
                temp = 0
                for index, row in fullObj.iterrows():
                    if ((row[column] == uniqueVal) and (row[lastColumn] == unique)):
                        temp += 1
                attributeDict[column][uniqueVal][unique] = temp
    for i in attributeDict:
        for j in attributeDict[i]:
            temp = 0
            for k in attributeDict[i][j]:
                temp += attributeDict[i][j][k]
            attributeDict[i][j]["Total"] = temp      
    for i in attributeDict:
        for j in attributeDict[i]:
            temp = 0
            for k in fullObj.iloc[:,-1].unique():
                if ((attributeDict[i][j][k]) == 0):
                    temp -= 0
                else:
                    temp -=  (((attributeDict[i][j][k]) / (attributeDict[i][j]["Total"]))) * math.log2(((attributeDict[i][j][k]) / (attributeDict[i][j]["Total"]))) 
            attributeDict[i][j]["Entropy"] = temp   
    fullEntropy = entropyCalculation(fullObj)
    for i in attributeDict:
        temp = fullEntropy
        for j in attributeDict[i]:
            temp -= (attributeDict[i][j]["Total"]) / (len(fullObj)) * attributeDict[i][j]["Entropy"]
        attributeDict[i]["Gain"] = temp
    return (attributeDict)
